# bpay

Unified Python SDK for Bangladeshi payment gateways.

## Features

- bKash integration
- Nagad integration
- SSLCommerz integration
- Unified payment abstraction
- Typed responses
- Async support

## Installation

```bash
pip install bpay
```

## Quick Start

```python
from bpay import BPay

client = BPay(provider="bkash")
```
